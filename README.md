# odin-recipes

This project is part of The Odin Project Foundations curriculum.

The goal of this project is to build a basic recipe website using HTML. The website will contain a main index page that links to several recipe pages.

By completing this project, I will have demonstrated the ability to:

* Create and organize HTML files
* Use headings, paragraphs, lists, and links
* Structure a simple website with multiple pages
* Work with images in HTML
* Use Git and GitHub for version control
* Clone repositories, commit changes, and push code to GitHub
